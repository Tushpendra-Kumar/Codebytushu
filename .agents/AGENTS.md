# Project Rules

- **Git Version Control**: Always commit (`git commit -m "..."`) and push (`git push`) your code changes when you complete a feature or finish fixing an issue. The user expects the AI agent to automatically push changes to GitHub so they can be immediately tested on their live server (Hostinger) without them needing to do it manually. 

- Note: If `git push` hangs or prompts for credentials, you may need to inform the user to push manually from their VS Code or GitHub Desktop. But always try to run `git push` automatically as part of your task completion process.
