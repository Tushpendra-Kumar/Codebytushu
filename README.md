## 🧠 About

**CodeByTushu** is a personal web project created by **Tushpendra Kumar**, a passionate **MERN Stack Developer and Linux Enthusiast**.  
The website is designed to showcase projects, coding tutorials, and creative ideas in web development and automation.  
It’s built for learners, coders, and creators who love exploring modern tech with simple, practical examples.

---

## ✨ Features

- 💻 Responsive and clean UI  
- ⚙️ Showcases real development projects  
- 🧩 Coding tutorials and practical examples  
- 📚 Beginner-friendly resources  
- 📬 Contact and connect options  

---

## 🏗️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript  
- **Backend:** Node.js / Express (for future updates)  
- **Database:** MongoDB  
- **Hosting:** Deployed via GitHub Pages / Custom Domain

## 🚀 Setup

1. Clone the repository  
   ```bash
   git clone https://github.com/yourusername/codebytushu.com.git
````

2. Open the folder

   ```bash
   cd codebytushu.com
   ```
3. Run the project

   * For static version → open `index.html` in a browser
   * For MERN setup → install and run

     ```bash
     npm install
     npm start
     ```
